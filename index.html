Ordenação de Algoritmos

Os algoritmos de ordenação são um conjunto de instruções que recebem um array ou lista como entrada e organizam os itens em uma ordem específica.

Bubble Sort

A ideia é percorrer o vetor diversas vezes, a cada passagem fazer flutuar para o topo o maior elemento. Assim como, bolhas em um tanque d'água

Selection Sort

É um algoritmo de ordenação baseado em se passar sempre o menor valor do vetor para a primeira posição, depois o de segundo menor valor para a segunda posição, e assim é feito sucessivamente.

Insertion Sort

Insertion Sort, ou ordenação por inserção, é um algoritmo de ordenação que, constrói uma matriz final com um elemento de cada vez, uma inserção por vez. É bastante eficiente para problemas com pequenas entradas.

Merge Sort

Consiste em dividir (o problema em vários subproblemas e resolver esses subproblemas através da recursividade) e Conquistar conquista é a união das resoluções dos subproblemas.

Quick Sort

Consiste em rearranjar as chaves de modo que as chaves "menores" precedam as chaves "maiores". Depois o quicksort ordena as duas sublistas de chaves menores e maiores recursivamente até que a lista completa se encontre ordenada.

Shell Sort

O algoritmo passa várias vezes pela lista dividindo o grupo maior em menores. Nos grupos menores é aplicado o método da ordenação por inserção.

Cocktail Sort

É uma variante do bubble sort, efetua ordenação por comparação. O algoritmo difere de um bubble sort no qual ordena em ambas as direções a cada iteração sobre a lista.

Radix Sort

O radix sort começa do dígito menos significativo até o mais significativo, ordenando assim: chaves curtas vem antes de chaves longas, e chaves de mesmo tamanho são ordenadas usando uma estrutura de ordem natural do produto cartesiano.

Heap Sort

Ordena os elementos à medida que insere na estrutura. No final das inserções, os elementos podem ser tirados da raiz da heap, na ordem desejada. Para uma ordenação decrescente, deve ser construída uma heap mínima. Para uma ordenação crescente, deve ser construído uma heap máxima.
